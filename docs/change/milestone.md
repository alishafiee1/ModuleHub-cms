## ✅ M3 — اجرا و مدیریت ماژول (Start/Stop)
**درصد اتمام: ۱۰۰%**
تاریخ شروع: — | تاریخ پایان: —

**توضیح:** ماژول نه فقط نصب میشه، بلکه **زنده میشه**! دکمه Start ماژول رو توی سرور اجرا می‌کنه — هم ماژول‌های معمولی (Node/Python) هم ماژول‌های Docker. محدودیت CPU و RAM هم داریم که ماژول دیوونه نشه و همه منابع سرور رو نخوره. وضعیت کارت هم رنگی میشه: سبز = روشنه، خاکستری = خاموشه، قرمز = کرشه کرده.

## ✅ M4 — کش هوشمند پکیج‌ها
**درصد اتمام: ۱۰۰%**
تاریخ شروع: — | تاریخ پایان: ۱۴۰۳/۰۳/۰۷

**توضیح:** ببین، وقتی یه ماژول نصب میشه، وابستگی‌هاش (مثلاً کتابخونه‌های Node یا Python) دانلود و ذخیره میشن. دفعه بعد که همون وابستگی‌ها لازم بشه، از کش استفاده میشه — نه دانلود دوباره. مثل اینه که وقتی یه چیز می‌خری و نگه‌اش می‌داری تا دوباره لازم نشه بری بخری. سرعت نصب دومی خیلی بیشتره.
