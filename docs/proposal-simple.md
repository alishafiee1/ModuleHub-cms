# پروپزال: ModuleHub CMS (نسخه ساده)

**سایتی که مثل یک کمد مرتب است — هر بخش جدا، هر چیز جدید بدون به‌هم‌ریختن بقیه اضافه می‌شود**

> جزئیات فنی: [design.md](design.md) · نسخه کامل: [proposal.md](proposal.md) · فهرست: [README.md](README.md)

---

## این پروژه برای چه کسی است؟

فرض کنید شما یک طراح وب، توسعه‌دهنده یا صاحب کسب‌وکار هستید. می‌خواهید سایتی داشته باشید که:

- نمونه‌کارها، آموزش‌ها، ویدیوها، عکس‌ها و خبرها را به مخاطب نشان دهد.
- همه‌چیز مرتب و دسته‌بندی‌شده باشد.
- هر وقت بخواهید بخش تازه‌ای اضافه کنید — گالری، بازی، داشبورد، صفحهٔ آموزشی — **لازم نباشد کل سایت را از نو بسازید یا کد اصلی را دست بزنید**.
- هر بخش برای خودش مستقل بماند و کار یک بخش، بقیه را خراب نکند.

**ModuleHub** همان «هستهٔ سایت» است: جایی که کارت‌ها را می‌چیند، مسیرها را نشان می‌دهد، و هر ماژول را جدا نگه می‌دارد.

---

## پوشه‌بندی سایت — مثل Explorer ویندوز

تصور کنید سایت‌تان مثل **پنجرهٔ Explorer** است: پوشه داخل پوشه، تا هر جا که بخواهید عمیق‌تر شود.

این پوشه‌ها فقط **برای مرتب‌کردن ظاهر سایت** هستند — مثل برچسب روی قفسه. وقتی یک **گالری** یا **پروژه** واقعاً اضافه می‌کنید، سرور برای همان بخش یک پوشهٔ **مستقل** در `standalone-modules/<instance-id>/` می‌سازد.

> **الان (v2):** فقط لیست flat کارت‌ها در `/` — breadcrumb و پوشهٔ مجازی 🔜

---

## چه تجربه‌ای به مدیر سایت می‌دهد؟

### ۱. اضافه کردن چیز جدید

| انتخاب | یعنی چه |
|--------|---------|
| **پوشهٔ جدید** | فقط دسته در JSON — 🔜 |
| **آپلود پروژهٔ خودم** | ZIP → پوشه مستقل + Docker برای تنظیمات |
| **انتخاب از آماده‌ها** | کپی از `core/catalog-modules/` → instance مستقل — 🔜 |

#### مسیر ZIP

آپلود → پوشه مجزا روی سرور → Docker برای **تنظیمات** → فرم (پورت، منابع، مسیر، GitHub، …). اگر تنظیمات ناقص بماند، کلیک روی کارت در admin دوباره فرم را باز می‌کند.

#### مسیر آماده‌ها

گالری یا مارک‌داون: سرور یک **نسخهٔ مخصوص سایت شما** می‌سازد — نام کارت، تصویر، رمز (اختیاری).

---

### ۲. مدیریت روزمره

- **الان:** Start/Stop از homepage وقتی admin login کرده ✅
- **هدف:** چرخ‌دنده ⚙ روی کارت — GitHub، تنظیمات، ZIP جزئی 🔜

---

### ۳. بازدیدکننده

فقط **کارت‌ها و محتوا** — بدون Add و ⚙. breadcrumb 🔜.
