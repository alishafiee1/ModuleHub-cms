# rtl درست بنویس 

RTL roll
وقتی من (هوش مصنوعی) دارم به تو به زبان فارسی جواب میدم، و توی جملاتم کلمات انگلیسی مثل Playwright، E2E، SSH یا Trace Viewer دارم، باید کل متن رو توی یه تگ <div> با مشخصات dir="rtl" و style="text-align:right;" قرار بدم.

این کار باعث میشه: ۱. کل متن از راست به چپ (RTL) باشه. ۲. ترازبندی (Alignment) متن هم راست باشه. ۳. کلمات انگلیسی توی مرورگر درست و بدون بهم ریختگی کنار کلمات فارسی دیده بشن.


```html
<div dir="rtl" style="text-align:right;">

متن چت منو تو

</div>
```
